Basico do HTML

Sempre fechar com </(e o comando); Ex: <h1> TEXTO TESTE</h1>
<h1-6> = Seria o cabeçalho, ou um titulo no site, quanto maior o numero depois do <h> menor sera o tamanho da letra
<br> = Usado para pular uma linha
<p> = usado para definir um paragrafo
<hr> = Uma linha horizontal
<b> ou <strong>(sendo esse o mais recomendado) usado para colocar o texto em negrito, <i> ou <em> (sendo este o mais recomendado) usado para colocar o texto em italico, <u> usado para criar uma underline em baixo de letras, <strike> usado para colocar uma linha no meio das letras, como se desse um sentido de errado
<ul> = Lista não ordenada, para mudarmos o tipo colocamos <ul type="E o tipo", tipos: circle, square, disc
<ol> = lista ordenada, para mudarmos o tipo colocamos <ol type="E o tipo", tipos: 1, A, a, I, i
<li> = Para adicionar itens a lista
<!-- --> = para adicionar um comentario no codigo  
img src="" = usado para inserir imagens, onde dentro do src colocamos o local da imagen/gif, seja ela por link de internet ou local no computador

<a href=""></a> = usado para colocar um link em uma pagina, onde depois do href colocamos o link do local do site, e dentro do >colocamos o nome que ira apareer no site<a
<table> = usado para abrir uma tabela
<table border="5"> = usamos este comando para definir a borda da tabela
    <table border="1"(sendo o tamanho da borda) width=""(sendo a largura)>  ---> estrutura de uma tabela, no width podemos tanto usar os numeros como pixels, ou usar em %, que correspondera ao tamanho no navegador
    <tr>                ---> onde o tr e a linha da tabela, ou seja quantos mais tr, mais linhas uma tabela podera conter
        <td> aaa </td>  ---> e o td e o que tem dentro da tabela, ou seja o conteudo, se adicionarmos colspan depois do td, ela ira 'comer' uma linha na lateral, se adicionarmos rowspan depois do td, ele ira 'comer' a linha de baixo
    </tr>                   ↳ Se substituirmos o td por th, vamo ter o cabeçalho da lista, ou seja, ela vai ficar em negrito e destacada
    </table>
<form> = usado para abrir um formulario, se adicionarmos um action depois do form, ele ira mandar para uma pagina setada
 <form> = estrutura de um formulario
        Login:<br>  ----->Nome escrito em cima 
        <input type="text" name="login"> -----> temos o tipo do formulario, e o nome para usarmos durante o codigo. 

    </form>
Tipos de formulario: text(texto padrão), password(usado em senhas, onde podemos ocultar a senha), button(botao), submit(usado para enviar informações), radio(como se fosse para escolher algo em bolinha), checkboc(escolher algom em quadrado)
↳ se adicionarmos o value depois do type, poderemos escolher oque vai estar dentro do formulario, usado mais na função do botão e o submit
<select> = usado para escolher alguns itens em listagem, (exemplo nos estados)
<select name="estado">
            <option value="sp">Sp</option>
            <option value="rj">Rj</option>
            <option value="sc">Sc</option>
            
</select>
<textarea = usado para o usuario colocar algum texto(Exemplo: alguns sites usam como obsevacao)
<textarea name="observacoes" rows="10" cols="50"></textarea> = sendo rows= a altura, e o cols a largura da caixa
se colocarmos o align no codigo, escolhemos o alinhamento(podendo ser right, left. center)
Se colocarmos background no codigo, escolhemos o fundo, podendo ser imagem ou cor
